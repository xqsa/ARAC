# ARAC Structure Notes



## Directory Map

- `docs/`: method, boundaries, and schemas.
- `src/arac/`: minimum Python package skeleton.
- `configs/`: frozen experiment contracts.
- `experiments/`: future runnable experiment folders.
- `data/`: raw, interim, processed data placeholders.
- `results/`: generated outputs only.
- `references/`: source pointers back to the original repository.
- `tests/`: smoke tests for the clean scaffold.

## Excluded By Design

- old Mxx milestone history
- Route A repair-chain artifacts
- cache/materialization outputs
- historical failed pilot directories
- HCC-specific problem special cases
